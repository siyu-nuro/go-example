### Getting Started
Make sure you have Go installed and $GOROOT and $GOPATH set up.

Install dependencies:
```
make dependencies
```

Build:
```
make build
```

Format:
```
make fmt
```

Run tests:
```
make test
```

Update dependencies:
```
glide update
```

Install:
```
make install
```

Run:
```
go-example --config config/production.yaml
```
